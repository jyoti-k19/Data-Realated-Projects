# 🪔 Diwali Sales Analysis  

## 📌 Project Overview  
This project analyzes Diwali sales data to uncover customer purchasing behavior and provide actionable business insights. The goal is to help businesses improve marketing strategies and increase revenue using data-driven decisions.

---

## 🎯 Objectives  

- Understand customer demographics and buying patterns  
- Identify top-performing product categories  
- Analyze sales trends across states and occupations  
- Generate insights for targeted marketing  

---

## 📂 Dataset  

The dataset contains information such as:  
- Customer ID  
- Gender  
- Age Group  
- Marital Status  
- State  
- Occupation  
- Product Category  
- Purchase Amount  

---

## 🔍 Key Steps Performed  

### 1️⃣ Data Cleaning  
- Removed null values  
- Dropped irrelevant columns  
- Corrected data types  

### 2️⃣ Exploratory Data Analysis (EDA)  
- Analyzed gender-wise and age-wise purchasing trends  
- Compared sales across states and occupations  
- Identified high-revenue product categories  

### 3️⃣ Data Visualization  
- Bar charts, count plots, and histograms  
- Insights derived using visual storytelling  

---

## 📊 Key Insights  

- 👩 **Married women (age 26–35)** are the highest buyers  
- 🏙️ Certain states contribute significantly to total revenue  
- 🛍️ Specific product categories dominate sales  
- 💼 Customers from particular occupations spend more
